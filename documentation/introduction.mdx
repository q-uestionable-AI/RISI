## What is RISI?

RISI is a pre-experimental security research project about persistent AI-agent memory. It studies
whether authorized retrieval changes adaptive state and whether ordinary, truthful interactions can
make a legitimate critical memory unavailable at a later decision point.

The project is intentionally narrow. It does not treat generic memory poisoning, prompt injection,
resource exhaustion, or ordinary retrieval degradation as RISI or CRAF.

## Research status

RISI, RISI-C, CRAF, and memory eclipsing are proposed terms and hypotheses. They are not validated
findings. Public claims will remain evidence-labeled and conditional on deterministic controls,
causal ablations, and external reproduction.

The authoritative research charter is maintained separately in the private research vault. This
documentation describes released repository behavior and public research boundaries only.
